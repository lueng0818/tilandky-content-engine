# Tilandky 爆文煉金系統 (Internal Tool)

這是 Tilandky 品牌的內部內容生產工具，基於 T.R.U.S.T. 系統設計。

## 功能模組
1. **礦場挖掘 (Source)**：輸入客戶痛點與關鍵字。
2. **濾鏡加工 (Reframe)**：套用工程師、資安、男性視角濾鏡。
3. **內容定位 (Categorize)**：確定貼文戰略目的 (漲粉/轉化)。
4. **標題工程 (Viral Titles)**：自動生成符合公式的爆文標題。
5. **變現導購 (CTA)**：自動生成導流至 $200 診斷的文案。

## 如何使用
1. 安裝環境：`pip install -r requirements.txt`
2. 啟動系統：`streamlit run app.py`
